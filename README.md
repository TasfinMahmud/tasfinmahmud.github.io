Personal portfolio website — CS Student | MERN Developer | AI Fintech Researcher

## 🚀 Deploy to GitHub Pages

1. Create a new repository on GitHub named exactly: `tasfinmahmud.github.io`
2. Push this folder:
   ```bash
   cd tasfinmahmud.github.io
   git init
   git add .
   git commit -m "Initial portfolio site"
   git branch -M main
   git remote add origin https://github.com/TasfinMahmud/tasfinmahmud.github.io.git
   git push -u origin main
   ```
3. Go to **Settings → Pages** → Source: `main` branch, root `/`
4. Your site will be live at **https://tasfinmahmud.github.io** within a few minutes!
